This is a snapshot of the ZPUino RetroCade Variant.

The Gadget Factory branch of the ZPUino RetroCade Source code is located at:
https://github.com/GadgetFactory/ZPUino-HDL
or
https://github.com/GadgetFactory/Papilio-SOC

Alvaro's official ZPUino repository is at:
https://github.com/alvieboy/ZPUino-HDL

To load the ZPUino RetroCade variant install Xilinx ISE Webpack and open:
zpuino\boards\papilio-pro\S6LX9\variants\retrocade\ise\ise.xise
zpuino\boards\papilio_one\s3e500\variants\retrocade\ise\ise.xise

or:
install cygwin and run make in:
zpuino\boards\papilio-pro\S6LX9\variants\retrocade\
zpuino\boards\papilio_one\s3e500\variants\retrocade\
